Garbage Detection System

This project involves the development of a garbage detection system aimed at identifying and classifying waste materials. It utilizes a dataset containing images of different types of garbage, such as plastic, metal, glass, and paper.

The system is built using a deep learning model that was trained to recognize various types of waste, helping to improve recycling processes and reduce littering in public areas.
